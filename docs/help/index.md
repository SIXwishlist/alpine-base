
# Usage


### Get
```shell
$ docker pull daspanel/alpine-base:latest
```

### Run
```shell
$ docker run --name=my-alpine-base daspanel/alpine-base:latest
```

### Stop
```shell
$ docker stop my-alpine-base
```

### Update image
```shell
$ docker stop my-alpine-base
$ docker pull daspanel/alpine-base:latest
$ docker run --name=my-alpine-base daspanel/alpine-base:latest
```
